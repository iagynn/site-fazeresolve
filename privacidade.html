Política de Privacidade

Última atualização: 16 de setembro de 2025

A sua privacidade é importante para nós. Esta política de privacidade explica quais dados pessoais a "Faz & Resolve" recolhe dos seus utilizadores e como esses dados são utilizados, em conformidade com as leis de proteção de dados.

1. Dados Recolhidos dos Nossos Clientes (Prestadores de Serviços)

  • Informações de Registo e Perfil: Recolhemos nome, e-mail, palavra-passe (armazenada de forma segura e encriptada), nome da empresa, documento (CNPJ/CPF), telefone e endereço para criar e gerir a sua conta.
  • Dados de Pagamento da Assinatura: Os detalhes do seu plano são geridos por nós. No entanto, os dados de cartão de crédito são processados diretamente pelo nosso parceiro de pagamentos, Mercado Pago, e não são armazenados nos nossos servidores, garantindo a sua segurança.
  • Dados de Integração: Com o seu consentimento, armazenamos tokens de autorização seguros para serviços de terceiros, como Google Calendar, Meta (WhatsApp) e Focus NFe, para ativar as funcionalidades de automação.

2. Dados que os Nossos Clientes Inserem na Plataforma (Dados dos Clientes Finais)

Os nossos clientes (prestadores de serviços) inserem e gerem na plataforma informações sobre os seus próprios clientes. Estes dados podem incluir nome, telefone, e-mail, endereço e o histórico de serviços prestados. Nestes casos, a "Faz & Resolve" atua como um processador de dados, agindo em nome do prestador de serviços, que é o controlador desses dados.

3. Como os Dados são Utilizados

  • Para Fornecer o Serviço Principal: Todos os dados recolhidos são essenciais para permitir que o prestador de serviços gira os seus pedidos, agendamentos, finanças e clientes de forma centralizada.
  • Para Comunicação: Utilizamos os dados de contacto (como o telefone do cliente final) para enviar notificações automáticas via WhatsApp (orçamentos, agendamentos, etc.), sempre em nome e sob a instrução do prestador de serviços.
  • Para Análises da Plataforma: Dados agregados e anonimizados são utilizados para alimentar funcionalidades como o painel de "Visibilidade de Mercado", ajudando o prestador a entender melhor o seu negócio, sem expor dados individuais.

4. Partilha de Dados com Terceiros

Para fornecer uma experiência completa, a nossa plataforma integra-se com serviços de terceiros. A partilha de dados ocorre apenas quando necessário para a funcionalidade correspondente:

  • OnRender: O nosso fornecedor de cloud, onde a aplicação e a base de dados estão hospedadas com segurança.
  • Meta (WhatsApp): Para o envio e recebimento de mensagens automáticas.
  • Mercado Pago: Para processar os pagamentos das assinaturas de forma segura.
  • Google: Para a sincronização de eventos com o Google Calendar.
  • Focus NFe: Para a emissão de notas fiscais, quando solicitado pelo utilizador.
  • ViaCEP: Para a consulta e preenchimento automático de endereços a partir do CEP.

Contacto
